# フロントエンド (React + TypeScript) の動かし方

このプロジェクトのフロントエンドは React + TypeScript (Vite) で構築されています。

## 前提条件

*   **Node.js**: 最新の安定版（LTS）がインストールされていること。
    *   確認コマンド: `node -v`

## セットアップ（初回のみ）

プロジェクトの依存関係（ライブラリなど）をインストールします。
`frontend` ディレクトリで以下のコマンドを実行してください。

```bash
cd frontend
npm install
```

## 開発サーバーの起動

開発中は以下のコマンドでサーバーを起動します。
ファイルの変更を検知して、ブラウザが自動的に更新されます。

```bash
cd frontend
npm run dev
```

起動後、ターミナルに表示される URL（例: `http://localhost:5173`）をブラウザで開いてください。

## 本番用ビルド

アプリを公開するために最適化されたファイルを生成する場合に使用します。
生成されたファイルは `dist` ディレクトリに出力されます。

```bash
cd frontend
npm run build
```

## プレビュー

ビルドされたアプリ（`dist` ディレクトリの内容）をローカルで確認する場合に使用します。

```bash
npm run preview
```
